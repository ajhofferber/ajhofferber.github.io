Greenhaus

Greenhaus is an application that allows users to better understand and manage their houseplants. Using Ruby on Rails and the Twilio API, users can not only search the database to find out more information about over 100 common houseplant types, but they can also sign up to receive text message reminders to water their plants. Each plant has it's own specific water index associated to it, and users will only receive a text when a particular plant needs some watering!

soundtrak

soundtrak allows users to search by movie title and listen to all of the tracks that are included in the film. Using Ruby on Rails for the backend and Backbone.js & Materialize for the from end, as well as the Spotify API & OMDB API, users can enter the title of a film and, using multiple ajax calls, quickly return a brief synopsis of the film, as well as a collection of playable Spotify tracks from the film. Additionally, if the user has an account, he or she is able to "favorite" a song and have it saved to their profile.

WaiterAid

WaiterAid is a restaurant management application, designed to help waitstaff, kitchen staff, and restaurant managers alike. It uses Ruby with a Sinatra framework, Javascript, jQuery, HTML & CSS. It is designed to be both mobile and desktop friendly.
